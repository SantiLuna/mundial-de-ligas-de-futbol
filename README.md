# Mundial de Ligas de Futbol

Prototipo frontend de una copa ficticia entre ligas de futbol. La experiencia permite explorar grupos, simular partidos, recorrer llaves de eliminacion directa y abrir plantillas por liga.

## Estado del proyecto

Este proyecto es un prototipo no oficial, creado con fines conceptuales, educativos y de demostracion. No esta afiliado, patrocinado ni aprobado por FIFA, UEFA, CONMEBOL, confederaciones, ligas, clubes, jugadores ni titulares de marcas.

Los nombres, logos, escudos, banderas y referencias deportivas se usan como identificadores visuales del prototipo. Cada marca, logo y nombre pertenece a su titular correspondiente. Los resultados, rankings, cruces y plantillas son datos de simulacion y no representan informacion oficial ni predicciones.

## Que incluye

- Landing de la iniciativa.
- Fase de grupos con simulacion de partidos.
- Clasificados directos y mejores terceros.
- Llaves orbitales con caminos hacia la final.
- Simulacion de eliminatorias.
- Plantillas por liga con titulares, suplentes y banderas por nacionalidad.

## Como ejecutarlo

No requiere build ni dependencias.

```powershell
python -m http.server 5173 --bind localhost
```

Abrir:

```text
http://localhost:5173/
```
